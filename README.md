# Peter Wanna Know (peter-wanna-know)

> 本專案 fork 自 [mvanhorn/last30days-skill](https://github.com/mvanhorn/last30days-skill)

這是 Peter 專用的 AI Agent 深度研究技能。它不依賴 SEO，而是直接從「人」所在的社群平台抓取過去 30 天的真實風向與數據，並自動合成結構化的研究報告。

## 核心特色

1.  **台灣社群整合**：特別加入 **PTT** 與 **Dcard** 追蹤，完整覆蓋台灣在地社群觀點。
2.  **全平台覆蓋**：並行檢索 Reddit, X (Twitter), YouTube, Hacker News, Polymarket, GitHub, TikTok, Instagram 等。
3.  **真金白銀的數據**：整合 Polymarket 預測市場，用真錢投票的賠率取代專家預測。
4.  **智能預研究**：自動解析實體（人、公司、產品），自動尋找對應的 X 帳號、GitHub Repo 與 Subreddit。
5.  **本地優先**：MIT 授權，所有研究數據與金鑰保存在本地，保護隱私。

## 安裝方式

在終端機執行：

```bash
npx skills add osakihsieh/peter-wanna-know -g -y
```

## 使用範例

在支援的 Agent 環境（如 Claude Code, Codex, Cursor）輸入：

```bash
/peter-wanna-know NVIDIA 營收社群反應
/peter-wanna-know 台灣股市近期熱點
/peter-wanna-know OpenClaw vs Hermes Agent 深度比較
```

## 配置要求

- **Python 3.12+**
- **Web Search Key** (推薦 Brave 或 Serper)，用於支援 PTT/Dcard 與網頁檢索。
- 其他選配：`SCRAPECREATORS_API_KEY` (TikTok/IG), `GITHUB_TOKEN` (GitHub 深度分析)。
